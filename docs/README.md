## What is ClawNeo?

ClawNeo is a personal AI assistant focused on a narrow but practical path:

- Discord as the main interaction channel
- OpenAI Codex OAuth for model access
- Local tools for reading files, searching code, and running shell commands
- Persistent user preferences and transcripts
- A lightweight local status UI and CLI service commands

## Platform support

- macOS: supported
- Linux: experimental
- Windows: not supported yet
