# 快速开始

本页帮助你用最短路径把 ClawNeo 跑起来。

## 1. 安装

```bash
npm install -g clawneo
```

## 2. 配置

需要时可运行交互式配置：

```bash
clawneo config
```

如果想看完整配置说明，请前往 [配置说明](/zh/guide/configuration)。

## 3. 启动服务

```bash
clawneo start
```

## 4. 打开本地状态 UI

服务启动后，打开：

```text
http://127.0.0.1:3210
```

## 5. 在 Discord 中使用 ClawNeo

ClawNeo 同时支持普通文本 `/xxx` 消息和原生 Discord 斜杠命令。

常用示例：

```text
/help
/status
/cancel
/update
/restart
/stop
```

## 6. 试几个自然语言任务

示例：

```text
20 分钟后提醒我检查日志
我有哪些提醒
```
