> ClawNeo 是一个面向“简单但真实可用”工作流的个人 AI 助手 CLI：本地运行，以 Discord 为主入口，并提供一组足够实用的内置工具。

## 产品预览

![ClawNeo 产品预览](/assets/clawneo-hero.jpg)

ClawNeo 把三部分连接成一条完整工作流：

- **Discord 对话**：负责自然交互与日常控制
- **本地运行时**：负责在你的机器上可控地执行任务
- **内置工具集**：覆盖文件、代码、Shell、技能与提醒等实际需求

## 为什么它更像一个产品，而不只是一个 CLI

### 你已经熟悉的入口
Discord 就是前台。你可以直接对话、触发系统命令、使用斜杠命令，而不必在多个控制面板之间来回切换。

### 本地执行，边界清晰
ClawNeo 不追求做成庞杂的平台，而是刻意把能力收敛在一组更容易理解、也更容易信任的内置工具上。

### 长期上下文真正可用
偏好、用户资料和长期上下文可以跨会话保留，同时 `USER.md` 仍然是人类可直接编辑的纯文本文件。

## 你通常可以用 ClawNeo 做什么

- 查看项目文件、配置与文档内容
- 在代码库中搜索符号、错误与引用关系
- 执行本地 shell 命令，处理开发或个人工作流任务
- 在需要额外能力时安装聚焦型技能
- 直接从 Discord 创建提醒式定时任务
- 管理本地服务并打开状态 UI
