> ClawNeo is a personal AI assistant CLI designed for people who want a simple setup: local runtime, Discord as the main interface, and a small set of tools that are actually useful every day.

## Product preview

![ClawNeo hero preview](/assets/clawneo-hero.jpg)

ClawNeo connects three pieces into one workflow:

- **Discord conversation** for natural interaction
- **Local runtime** for controllable execution on your machine
- **Built-in tools** for files, code, shell, skills, and reminders

## Why it feels like a product instead of just a CLI

### One interface you already use
Discord becomes the front door. You can talk to ClawNeo naturally, trigger system commands, and use slash commands without switching between multiple control panels.

### Local execution with practical boundaries
ClawNeo keeps the tool surface intentionally narrow. Instead of trying to be a giant platform, it focuses on a small set of built-in capabilities that are easier to trust and reason about.

### Persistent context that stays useful
Preferences, profile data, and durable user context can be kept across sessions, including a plain-text `USER.md` file that remains editable by humans.

## Typical things you can do with ClawNeo

- inspect project files and configs
- search a codebase for symbols, errors, or references
- run shell commands for local tasks and development workflows
- install focused skills when you need extra capability
- create reminder-style scheduled tasks directly from Discord
- manage the local service and open the status UI
