# 命令使用

本页汇总了本地运行 ClawNeo 和在 Discord 中使用时最常见的命令。

## CLI 服务命令

| 命令 | 说明 |
| --- | --- |
| `clawneo start` | 启动本地 ClawNeo 服务 |
| `clawneo stop` | 停止正在运行的服务 |
| `clawneo restart` | 重启服务 |
| `clawneo status` | 查看服务状态 |
| `clawneo ui` | 打开本地状态 UI |
| `clawneo config` | 打开交互式配置 |

## 基础 CLI 参数

| 命令 | 说明 |
| --- | --- |
| `clawneo -h` | 查看帮助 |
| `clawneo --help` | 查看帮助 |
| `clawneo -v` | 查看版本 |
| `clawneo --version` | 查看版本 |

## Discord 系统命令

ClawNeo 同时支持普通文本 `/xxx` 消息和原生 Discord 斜杠命令。

| 命令 | 说明 |
| --- | --- |
| `/help` | 查看可用系统命令 |
| `/status` | 查看服务或机器人状态 |
| `/cancel` | 中止当前 Discord 会话里正在执行的任务 |
| `/update` | 在后台更新 ClawNeo 并重启服务 |
| `/restart` | 重启服务 |
| `/stop` | 让机器人离线，直到再次启动 |
